CertLoop

プライバシーポリシー

最終更新: 2026年4月20日

1. はじめに

CertLoop(以下「本アプリ」)は、AWS 認定資格 (Solutions Architect - Associate) の合格を支援する iOS アプリケーションです。本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明します。

2. 収集する情報

本アプリは以下の情報を取得・利用します。

(a) ユーザーが入力する情報

  • 表示名(ニックネーム)
  • AWS 経験レベル(初学者・経験者など)
  • 受験予定時期
  • 1日あたりの学習時間目標

(b) 学習履歴

  • 各問題への回答(選択した選択肢、正誤)
  • 回答時刻、所要時間
  • 模試の受験結果(スコア、カテゴリ別正答率)
  • 学習ストリーク、累計学習日数

(c) 購入情報

StoreKit を通じてサブスクリプション(CertLoop Pro)の状態を管理します。購入処理は Apple Inc. が行い、クレジットカード情報等は開発者に共有されません。

3. データの保存場所

上記 2(a)(b) のすべてのデータは、ユーザーの端末内(SwiftData)にのみ保存されます。開発者が運営するサーバーへのデータ送信は行いません。

4. AI 機能とデータ送信(重要)

本アプリは AI 解説機能のため、Google Gemini API(Google LLC が提供)を使用しています。AI 機能を利用する際、以下のデータが Google のサーバーに送信されます。

  • 不正解だった問題 ID と問題文
  • あなたが選択した選択肢
  • 正解の選択肢
  • あなたの AWS 経験レベル(プロンプトの調整に使用)

送信されるデータには、氏名・メールアドレス等の個人を特定する情報は含まれません。Google による AI API の利用については、Google プライバシーポリシー および Gemini API 利用規約 をご確認ください。

AI 機能を使わない問題演習(自分で解いて答え合わせをする部分)は、データの外部送信なしで動作します。

5. 第三者サービス

本アプリは以下の第三者サービスを利用しています。

  • Google Gemini API(Google LLC) — AI 解説生成のために使用
  • Apple StoreKit(Apple Inc.) — サブスクリプション管理のために使用

広告 SDK やアナリティクス SDK は使用していません。

6. 通知

本アプリは学習リマインダーのため、ローカル通知(iOS の UserNotifications)を使用します。通知はあなたの端末内でスケジュールされ、外部サーバーは介在しません。通知の許可・拒否は iOS の設定からいつでも変更できます。

7. データの共有

ユーザーの個人情報を第三者に販売・提供することはありません。ただし、以下の場合を除きます。

  1. 法令に基づく開示要求があった場合
  2. ユーザーの同意がある場合

8. データの削除

アプリ内の設定画面から「全データをリセット」できます。また、アプリをアンインストールすることで、端末内に保存されたすべてのデータが完全に削除されます。一度削除したデータは復元できません。

9. お子様のプライバシー

本アプリは 13 歳未満のお子様を対象としていません。13 歳未満のお子様から意図的に個人情報を収集することはありません。

10. ポリシーの変更

本プライバシーポリシーは、必要に応じて変更される場合があります。変更後のポリシーはアプリ内に掲載した時点から効力を生じます。

11. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、yasudariku1214@gmail.com までご連絡ください。
