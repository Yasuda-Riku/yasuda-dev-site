FitLoop

プライバシーポリシー

最終更新: 2026年4月14日

FitLoop(以下「本アプリ」)は、ユーザーのプライバシーを最も重要な価値として設計されています。本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明します。

基本方針: あなたのデータは原則として端末の外に出ません。

第1条 収集する情報

本アプリは以下の情報を取得・利用します。

(a) ユーザープロフィール情報

ニックネーム、経験レベル(未経験/〜半年/半年〜2年/2年以上)、目標(筋肥大/ダイエット/健康維持/体力向上)、週のトレーニング日数をユーザーが入力した際に取得します。これらは端末内のみに保存されます。

(b) トレーニング予定・記録

週単位のトレーニング予定と、実際に記録したワークアウト結果(種目名、セット数、重量、反復回数、RPE)を端末内に保存します。外部サーバーには送信されません。

(c) Google Gemini API への送信データ

AI による週間プラン生成およびメニュー更新の目的で、以下の情報のみが Google Gemini API(Google LLC)に送信されます。

  • ニックネーム
  • 経験レベル・目標
  • 週のトレーニング日数
  • 先週のワークアウト記録(種目名、重量、回数、RPE)

個人を特定する情報、健康状態、医療履歴、財務情報、位置情報、連絡先、写真、音声などは一切送信されません。

(d) 通知設定データ

トレーニング日の朝 9 時に送信するローカル通知の設定情報は、端末内のみで処理されます。通知内容の生成・送信はすべて端末内で完結します。

(e) 購入情報

RevenueCat, Inc. を通じてサブスクリプションの状態を管理します。購入処理は Apple Inc. が行い、クレジットカード情報等は開発者に共有されません。

第2条 情報の利用目的

取得した情報は以下の目的にのみ使用します。

  1. Google Gemini API を利用した AI 訓練プラン生成
  2. ワークアウト記録の端末内での管理
  3. トレーニング日の通知送信(端末内処理)
  4. サブスクリプション状態の管理
  5. アプリの改善・不具合修正

第3条 データの保存場所

ニックネーム、トレーニング予定、ワークアウト記録、アプリ設定などすべてのユーザーデータは、ユーザーの端末内(Apple SwiftData)にのみ保存されます。

クラウドバックアップ、同期サーバー、または外部クラウドサービスへのデータ送信は原則として行いません(第1条(c)に記載の Gemini API への送信を除く)。

第4条 Google Gemini API の利用について

本アプリは Google Gemini API を利用して AI 訓練プラン生成機能を提供しています。

  • 通信は HTTPS による安全な接続で行われます
  • 送信データは Google LLC の規定に従って処理されます
  • Google Gemini API のプライバシーポリシーもご確認ください

第5条 第三者サービス

本アプリは以下の第三者サービスを利用しています。各サービスのプライバシーポリシーもご確認ください。

  • Google Gemini API(Google LLC) — AI による訓練プラン生成のために使用
  • RevenueCat, Inc. — サブスクリプション状態の管理のために使用
  • Apple UserNotifications — ローカル通知送信のために使用(端末内完結)

第6条 データの共有

ユーザーの個人情報を第三者に販売・提供することはありません。ただし、以下の場合を除きます。

  1. 法令に基づく開示要求があった場合
  2. ユーザーの明示的な同意がある場合

Gemini API への送信データについては、Google LLC のプライバシーポリシーに従って処理されます。

第7条 データの削除

以下の方法でデータを削除できます。

  1. アプリ内の「設定」タブ > 「WIPE ALL DATA」ボタン — 端末内の全データを即時削除
  2. アプリを端末から削除 — 端末内に保存されたすべてのデータが消去されます

第8条 お子様のプライバシー

本アプリは 4 歳以上を対象としていますが、お子様が本アプリを利用される場合は保護者の監督の下で使用してください。本アプリは、保護者の同意なしに 13 歳未満のお子様から個人情報を意図的に収集することはありません。

第9条 ポリシーの変更

本プライバシーポリシーは、必要に応じて変更される場合があります。変更後のポリシーは本ページおよびアプリ内に掲載した時点から効力を生じます。

第10条 お問い合わせ

本プライバシーポリシーに関するお問い合わせは、yasudariku1214@gmail.com までご連絡ください。
