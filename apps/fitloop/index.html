よくある質問

AI メニューが生成されません

メニュー生成にはインターネット接続が必要です(Google Gemini API を使用しています)。Wi-Fi またはモバイルデータ通信が有効か確認してください。一時的な API の混雑で失敗することがありますので、数分後に再試行してください。

データは外部に送信されますか?

AI メニュー生成時に、ニックネーム・経験レベル・目標・先週のワークアウト記録が Google Gemini API に送信されます。それ以外のデータ(健康情報、位置情報、連絡先など)は一切送信されません。ワークアウト記録やプロフィールは端末内にのみ保存されます。詳しくはプライバシーポリシーをご覧ください。

サブスクリプション(Pro)を解約したい

iOS の「設定 > あなたの名前 > サブスクリプション」から、いつでも解約できます。解約後も有効期間終了までは引き続き Pro 機能をご利用いただけます。無料トライアル期間中にキャンセルした場合、課金は発生しません。

アプリを削除するとデータはどうなりますか?

トレーニング記録・プロフィール・メニュー等のデータはすべて端末内に保存されています。アプリを削除するとそれらのデータは失われ、復元はできません。設定タブの「WIPE ALL DATA」で個別にデータを消去することもできます。

通知が届きません

iOS の「設定 > 通知 > FitLoop」で通知を許可してください。通知はトレーニング日の朝 9 時にローカルで送信されます。端末の「おやすみモード」や「集中モード」がオンの場合は届かないことがあります。

バグを見つけたら

お手数ですが、上記のメールアドレスまで「発生した状況」「iOS のバージョン」「端末モデル」を添えてご連絡ください。可能であればスクリーンショットも一緒にお送りいただけると助かります。
