利用規約

最終更新: 2026年4月2日

第1条(適用)

本利用規約(以下「本規約」)は、城印帳(以下「本アプリ」)の利用に関する条件を定めるものです。ユーザーは本アプリをダウンロードまたは利用することにより、本規約に同意したものとみなされます。

第2条(サービス内容)

本アプリは以下のサービスを提供します。

  1. GPS 位置情報を利用した城へのチェックイン機能
  2. デジタル御城印の生成・コレクション機能
  3. 登城記録の管理・エクスポート機能
  4. お気に入り城の管理・ウィジェット表示機能

サービス内容は予告なく変更・追加・廃止される場合があります。

第3条(アカウントと認証)

本アプリは Firebase 匿名認証を使用します。ユーザーが明示的にアカウントを作成する必要はありませんが、アプリを削除した場合、データが失われる可能性があります。

第4条(プレミアムサブスクリプション)

  1. 本アプリは無料で基本機能を利用できますが、一部機能はプレミアムサブスクリプション(有料)への加入が必要です。
  2. サブスクリプションの購入は Apple ID を通じて行われ、Apple Inc. の利用規約に従います。
  3. サブスクリプションは有効期間終了日の 24 時間前までにキャンセルしない限り、自動的に更新されます。
  4. 更新料金は Apple ID アカウントに請求されます。
  5. サブスクリプションの管理・キャンセルは、iOS の「設定」>「Apple ID」>「サブスクリプション」から行えます。

第5条(広告)

無料ユーザーに対しては Google AdMob による広告が表示されます。リワード広告の視聴により一部機能(銀印のロック解除)が利用可能になります。広告の表示内容は広告配信事業者により決定され、本アプリの開発者はその内容について責任を負いません。

第6条(位置情報の利用)

  1. 本アプリはチェックイン機能のために端末の位置情報を使用します。
  2. 位置情報はチェックインの判定および登城記録の保存に使用し、外部サーバーへの送信は行いません。
  3. 位置情報の利用許可は iOS の設定からいつでも変更できます。

第7条(知的財産権)

  1. 本アプリのデザイン、コード、コンテンツに関する知的財産権は開発者に帰属します。
  2. ユーザーが取得したデジタル御城印の画像は、個人利用および SNS 等での共有に限り自由に使用できます。
  3. デジタル御城印を商業目的で使用することは禁止します。

第8条(禁止事項)

ユーザーは以下の行為を行ってはなりません。

  1. GPS の偽装やその他の不正手段によるチェックイン
  2. 本アプリのリバースエンジニアリング、逆コンパイル
  3. 本アプリの機能を妨害する行為
  4. 本アプリを利用した違法行為
  5. 取得したデータの不正な改ざん・再配布

  2. GPS 精度、ネットワーク状況等によりチェックインが正常に動作しない場合があります。
