最初の一行は「accumulator += dt;」。dt（経過秒数）をとにかく貯金していくイメージ。
1秒貯まったら使う。そのために while (accumulator >= DROP_SECONDS) { ... } で、溜まっている限り繰り返す。
ループの中では accumulator -= DROP_SECONDS; と blockRow++; を並べる。while にする理由はタブを切り替えて数秒分まとめて dt が来るケースに備えるため。
下に行き切った時に blockRow = 0 に戻すのを忘れずに。if (blockRow >= ROWS) blockRow = 0; の1行でOK。
