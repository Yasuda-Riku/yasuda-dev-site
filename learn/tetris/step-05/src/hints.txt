canMove は return true; の前に 4 つの「if (...) return false;」を並べるだけ。例: if (col < 0) return false;
update の落下は「if (canMove(blockCol, blockRow + 1)) blockRow++;」の 1 行に置き換え。if (blockRow >= ROWS) の行は不要になります。
onKey は各行に && canMove(...) を足す形。例: if (key == Key.LEFT && canMove(blockCol - 1, blockRow)) blockCol--;
