Course · 応用講義の前に

応用に向かう前に、
もう一度ここだけ。

Java 応用 (JDBC / Servlet / MVC) で「いきなり要求される」基礎技術を、 応用の文脈そのままの問題で再復習するミニコース。 基礎復習コースをやり込まなくても、ここだけでも応用の山場の半分以上は越えられます。

なぜこのコース?

応用で詰まるのは、
応用の知識じゃなく基礎。

JDBC で Connection conn = ... と書くとき、左辺の Connection は interface です。 SQLException を throws でリレーするのは checked 例外のお作法です。 extends HttpServlet + @Override doGet は 継承とアノテーションです。

つまり応用は「新しい知識」より「基礎が応用の文脈で組み合わさる」場面の連続。 このコースは、応用カリキュラムで実際に出てくる形そのままの問題で、必要な基礎だけを濃縮して復習します。
