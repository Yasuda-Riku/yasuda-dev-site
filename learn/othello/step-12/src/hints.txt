resetGame は board を全部 0 で埋めてから中央 4 マスに初期石、その他の状態 (currentPlayer / gameOver / cursor / aiCooldown) も初期値に戻す。
onKey では if (key == Key.R) { resetGame(); return; } を最初に書く。これで gameOver 中でも R が効く。
