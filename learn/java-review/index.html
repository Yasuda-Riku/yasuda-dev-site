Java 研修の
復習クイズ。

会社の Java 研修で習った内容を、未経験者が詰まりやすい順に復習できる短いクイズ集です。 間違えたら AI が「なぜそうなのか」を解説します。 進度はブラウザに保存されるので、毎日少しずつ進められます。

使い方

3 ステップ。

1. セクションを開いて、空欄に答えを入力 →「解答」を押す。
2. 正解なら次へ。間違いなら「ヒントを見る」、それでもわからなければ「AI に解説してもらう」。
3. 間違えた問題は明日以降「復習対象」として戻ってきます。

進捗はブラウザの localStorage に保存されます。サーバには送られません。 別の端末や別のブラウザでは別の進捗として扱われます。
