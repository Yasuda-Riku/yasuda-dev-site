Java 研修の
復習クイズ。

会社の Java 研修で習った内容を、未経験者が詰まりやすい順に復習できる短いクイズ集です。 間違えたら AI が「なぜそうなのか」を解説します。 進度はブラウザに保存されるので、毎日少しずつ進められます。

応用準備 (JDBC / Servlet / MVC 向け)

応用前にここだけ。

応用講義 (JDBC / Servlet / MVC) で「いきなり要求される」基礎を、応用そのままの文脈で復習する 7 セクション。 概念解説ページが付いているセクションは、問題の前に読むと理解が早いです。

セクション 1 · 7 問 interface を使いこなす JDBC で必須 — Connection / PreparedStatement / ResultSet
✓ セクション 2 · 7 問 例外処理 (応用版) SQLException · try-with-resources · throws リレー
✓ セクション 3 · 5 問 List/Map で結果を持ち回る List<Employee> · Map<String, String[]> · 拡張 for
✓ セクション 4 · 6 問 HttpServlet を継承する extends · @Override · doGet/doPost · req.getParameter
✓ セクション 5 · 4 問 アノテーション基礎 @WebServlet · @Override · メタ情報
✓ セクション 6 · 5 問 DTO / Bean 規約 private + getter/setter · 引数なしコンストラクタ
✓ セクション 7 · 5 問 セッションとキャスト HttpSession · getAttribute → ダウンキャスト

使い方

3 ステップで進める。

  1. 1

    解いてみる

    セクションを開いて、空欄に答えを入力し 「解答」 を押します。

  2. 2

    詰まったら頼る

    まずは 「ヒントを見る」。それでも分からなければ 「AI に解説してもらう」 で個別解説が出ます。

  3. 3

    まとめて復習

    間違えた問題は自動で 「間違えた問題」 セクションに集まります。正解すればそこから消えるので、空になるまでやれば一通り押さえたことに。

保存場所

進捗はこのブラウザの localStorage にだけ保存されます。サーバには送信されません。 そのため、別の端末・別のブラウザで開くと別の進捗として扱われます。
